## 封装axios

对于区分开发环境和生产环境，vite会使用.env 从我的环境目录中的下列文件加载额外的环境变量

~~~
.env # 所有情况都会加载
.env.local # 所有情况下都会加载，但会被git忽略 git ignore中有*.loacl
.env.[modal] # 只有在指定模式下加载
.env.[modal].local
~~~
